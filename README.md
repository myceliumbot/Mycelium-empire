# 🐍 Snake Game

A modern, responsive Snake game built with HTML5, CSS3, and JavaScript. Features beautiful graphics, smooth animations, and cross-platform compatibility.

## 🎮 Features

- **Modern UI**: Sleek design with gradient backgrounds and glassmorphism effects
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Multiple Controls**: 
  - Keyboard (Arrow keys or WASD)
  - Touch/swipe controls for mobile
  - On-screen buttons for mobile devices
- **Progressive Difficulty**: Game speed increases as you score more points
- **Score System**: Track your current score and high score (saved locally)
- **Pause Functionality**: Press Space to pause/resume the game
- **Visual Effects**: Glowing snake segments, animated food, and smooth transitions
- **Sound-Free**: Perfect for playing anywhere without disturbing others

## 🎯 How to Play

1. **Objective**: Control the snake to eat the red food dots
2. **Growth**: Each food eaten makes the snake longer and increases your score
3. **Avoid Collisions**: Don't hit the walls or your own tail
4. **Scoring**: Each food gives you 10 points
5. **Speed**: The game gets faster every 50 points

## 🕹️ Controls

### Desktop
- **Arrow Keys** or **WASD**: Move the snake
- **Space**: Pause/Resume game
- **Mouse**: Click buttons for game control

### Mobile
- **Swipe**: Swipe in any direction to move the snake
- **Touch Buttons**: Use the on-screen arrow buttons
- **Tap**: Tap buttons for game control

## 🚀 How to Run

1. **Simple Setup**: Just open `index.html` in any modern web browser
2. **No Installation Required**: No dependencies or build process needed
3. **Cross-Platform**: Works on Windows, macOS, Linux, iOS, and Android

## 🎯 Tips for High Scores

1. **Plan Ahead**: Think about where your snake will be in a few moves
2. **Use the Grid**: The subtle grid lines help with navigation
3. **Don't Rush**: Take your time, especially as the game speeds up
4. **Practice**: The more you play, the better you'll get at anticipating moves

## 🔧 Customization

The game is easily customizable by modifying the CSS variables and JavaScript constants:

- **Colors**: Change the color scheme in `style.css`
- **Speed**: Adjust game speed in `game.js`
- **Grid Size**: Modify the canvas and grid dimensions
- **Scoring**: Change point values and speed increase intervals
