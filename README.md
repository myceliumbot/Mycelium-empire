# Breakout Game

A simple Breakout game built with HTML5 Canvas and JavaScript.

## How to Play

1. Open `index.html` in a modern web browser.
2. Use the **Left** and **Right** arrow keys to move the paddle.
3. Break all the bricks without letting the ball fall below the paddle.
4. You have 3 lives. Good luck!

## Project Structure

- `index.html` – Main HTML file that sets up the canvas and includes scripts.
- `styles.css` – Basic styles for the page and canvas.
- `game.js` – Game logic and rendering.

Enjoy the game! :)
